# School Promo Motion

30-second Remotion promo built as a motion-graphics system over live-action footage.

## Run

```bash
npm install
npm run dev
```

Open Remotion Studio and preview the `SchoolPromoMotion` composition.

## Assets

The footage used by the composition lives in `public/kling/`:

- `kling_scene1.mp4`
- `kling_scene2.mp4`
- `kling_scene3.mp4`
- `kling_scene4a.mp4`
- `kling_scene4b.mp4`
- `kling_scene4c.mp4`
- `kling_scene5.mp4`
